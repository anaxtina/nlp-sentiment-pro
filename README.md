EN # 🧠 Tech Sentiment Analysis Logic

A specialized sentiment analysis engine focused on technical feedback. This module demonstrates the practical application of **Natural Language Processing (NLP)** and **Algorithmic Logic**.

## 🚀 Technical Highlights
- **NLP Pipeline:** Implements string normalization, punctuation removal, and tokenization.
- **Efficiency:** Optimized methods with $O(n)$ complexity analysis.
- **Scalability:** Capable of batch processing with automated statistical reporting.
- **Static Typing:** Leverages Python's `typing` module to ensure clean and maintainable code.

## 🛠️ How it Works
The algorithm uses a proprietary lexicon to assign weights to technical keywords (e.g., "bug" vs. "innovative"). It calculates a final score to classify the text as **Positive**, **Negative**, or **Neutral**.

## 🔬 Scientific Approach
By bridging **Machine Learning** foundations with **Human Behavior** analysis, this project aims to provide a more nuanced understanding of how technical users express satisfaction and friction in digital environments.



PT # 🧠 Tech Sentiment Analysis Logic

Este módulo é um motor de análise de sentimentos focado em feedbacks tecnológicos. Ele demonstra a aplicação prática de **Processamento de Linguagem Natural (NLP)** e **Lógica Algorítmica**.

## 🚀 Destaques Técnicos
- **Pipeline de NLP:** Implementa normalização de strings, remoção de pontuação e tokenização.
- **Eficiência:** Métodos otimizados com análise de complexidade $O(n)$.
- **Escalabilidade:** Capaz de processar lotes (batch analysis) e gerar relatórios estatísticos automáticos.
- **Tipagem Estática:** Uso de `typing` do Python para garantir código limpo e seguro.

## 🛠️ Como funciona?
O algoritmo utiliza um léxico proprietário para atribuir pesos a palavras-chave técnicas (ex: "bug" vs "innovative"), calculando um score final que classifica o texto como **Positive**, **Negative** ou **Neutral**.
