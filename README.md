# 🧠 Tech Sentiment Analysis Logic

Este módulo é um motor de análise de sentimentos focado em feedbacks tecnológicos. Ele demonstra a aplicação prática de **Processamento de Linguagem Natural (NLP)** e **Lógica Algorítmica**.

## 🚀 Destaques Técnicos
- **Pipeline de NLP:** Implementa normalização de strings, remoção de pontuação e tokenização.
- **Eficiência:** Métodos otimizados com análise de complexidade $O(n)$.
- **Escalabilidade:** Capaz de processar lotes (batch analysis) e gerar relatórios estatísticos automáticos.
- **Tipagem Estática:** Uso de `typing` do Python para garantir código limpo e seguro.

## 🛠️ Como funciona?
O algoritmo utiliza um léxico proprietário para atribuir pesos a palavras-chave técnicas (ex: "bug" vs "innovative"), calculando um score final que classifica o texto como **Positive**, **Negative** ou **Neutral**.
